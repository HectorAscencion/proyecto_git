# Git Ejemplo

Repositorio de ejemplo para practicar Git, ramas, merges y commits.

## Archivos del proyecto

- **main.py**: Archivo principal del proyecto.
- **funciones.py**: Contiene funciones matemáticas básicas.

## Historial de commits

| Commit | Mensaje |
| ------- | -------- |
| 1 | Creación inicial del proyecto |
| 2 | Agregado de funciones |
| 3 | Uso de ramas y merges |
| 4 | Comprobacion |
## Ramas

- **main**: Rama principal
- **rama1**: Rama secundaria para pruebas
- **rama2**: Rama comprobacion final

## Cómo usar

1. Clonar repositorio
2. Ejecutar el programa
3. Realizar cambios y commits
